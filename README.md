## 🛠️ Shortcut Library

### 📂 O - Open & Navigation
*Use these to launch apps or folders instantly.*

| Action | Shortcut |
| :--- | :--- |
| **O**pen File Explorer | `Win` + `E` |
| **O**pen Settings | `Win` + `I` |
| **O**pen Task Manager | `Ctrl` + `Shift` + `Esc` |
| **O**pen Search Bar | `Win` + `S` |
| **O**pen Run Command | `Win` + `R` |

---

### 📋 C - Close & Copy
*Manage your windows and data.*

| Action | Shortcut |
| :--- | :--- |
| **C**lose Active Window | `Alt` + `F4` |
| **C**lose Current Tab | `Ctrl` + `W` |
| **C**opy to Clipboard | `Ctrl` + `C` |

---

### 🖥️ S - System & Snap
*Control your workspace layout.*

| Action | Shortcut |
| :--- | :--- |
| **S**nap Window Left/Right | `Win` + `←` / `→` |
| **S**how Desktop | `Win` + `D` |
| **S**nipping Tool (Screenshot) | `Win` + `Shift` + `S` |

---

### 🔒 L - Lock & Layout
| Action | Shortcut |
| :--- | :--- |
| **L**ock PC | `Win` + `L` |
| **L**ayout (Virtual Desktops) | `Win` + `Tab` |

### 📋 V - View & Clipboard
